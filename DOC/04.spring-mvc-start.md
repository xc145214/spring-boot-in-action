# Spring MVC 基础

MVC: Model + View + Controller

三层架构： Presentaition  + Application + Data

Spring MVC 提供一个 DispatcherServlet 来开发 Web 应用：
+ 2.5 及以下需要配置 web.xml
+ 3.0 及以上通过实现 WebApplicationInitializer 接口来实现 web.xml 的配置。
